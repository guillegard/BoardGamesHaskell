# Reinas

Programa que encuentra solucion al problema de N reinas

Instrucciones:
Cargar archivo:   
	:l reinas.hs
Correr porgrama
	main n
	n = numero de reinas, tablero n x n

Ejemplo:
	:l reinas.hs
	main 8
